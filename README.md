# Load Balancing for Resource Optimization in Edge Computing

## Project Overview
This project uses Proximal Policy Optimization (PPO) to optimize load balancing and VM migration in an edge computing environment.

## Features
- Intelligent load balancing
- VM migration optimization
- Reduced delay
- Improved resource utilization
- PPO-based reinforcement learning

## Technologies Used
- Python
- PyTorch
- Reinforcement Learning (PPO)
- Edge Computing

## Project Structure
- main.py : Main execution file
- train_ppo.py : PPO training script
- ppo_agent.py : PPO agent implementation
- cloudlet_env.py : Environment definition
- cloudlet_simulator.py : Cloudlet simulation

## How to Run

1. Install dependencies
2. Run:

```bash
python main.py
```

## Results
The PPO agent learns optimal migration decisions to reduce delay and improve resource utilization in edge computing environments.
